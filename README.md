## Build the project

### Install dependencies

Install the [Angular CLI](https://cli.angular.io/) globally

```
npm install -g @angular/cli@9.1.1
```

Install required packages

```
npm install
```

### Run the build command

```
npm run build[:dev|:prod]
```
